# Weather-App

### Description
* Need city name.
* Passing City name through Visual Crossing Weather Api.
* Fetching data from Openweather.
* Showing it on the screen.
* Also display your current city weather condition.

### Technologies
* JSX
* CSS
* Visual Crossing Weather Api
* React
* React Hooks
